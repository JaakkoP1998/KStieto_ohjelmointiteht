# KSTieto ohjelmointitehtävä
Sisältää koodit KStiedon ohjelmointitehtävän Back- ja Front-endiin.
Kansiossa KSTieto on back-endin PHP-koodi. KSTietoFront sisältää vue.js koodit.

Back-end toteutettiin käyttäen XAMPP-ohjelmistopakettia.
Tietokantana käytettiin mySQL-tietokantaa. Tietokanta koostui kahdesta taulukosta, "users" ja "comments".
Front-end toteutettiin käyttäen nodea.
