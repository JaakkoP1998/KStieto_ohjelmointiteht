- Tehtävänannossa on käytetty XAMPP-pakettia PHP:n ja tietokannan luonnissa.
- Front-end on toteutettu nodella.
Muita työkaluja käytetty: 
- https://vuejs.org/guide/scaling-up/routing eri formien välillä vaihtelussa.
- ChatGPT CORS-asetuksien säädössä php-puolella.

SQL-suunnitelma:
CREATE TABLE Users (
  UserName varchar(255) PRIMARY KEY,
  PassWord varchar(255) NOT NULL,
  FOREIGN KEY (Content)
);


CREATE TABLE Comments (
  CommentID int PRIMARY KEY,
  Content varchar(255) NOT NULL,
  FOREIGN KEY (UserName)
);
